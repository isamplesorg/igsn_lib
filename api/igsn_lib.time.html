1.2. igsn_lib.timeÂ¶

This libary provides some convenience methods for working with Julian dates.

A Julian Date is a floating point value that identifies the number of days before or after -4713-11-24 12:00:00.000 UTC. The range is unlimited, for example the estimated age of the universe is about 13.787 billion years 1, and that date is far outside of the common operating parameters for databases and many programmatic representations of time.

In JD, the start of the universe can be reliably expressed as approximately:

import astropy.time
tstart = astropy.time.Time(-13.787E9, format='jyear')
jd = tstart.jd
print(f"Universe started on Julian Date: {jd}")

Universe started on Julian Date: -5035700028955.0

The start of the universe can not be expressed in a python datetime:

try:
  print(tstart.to_datetime(datetime.timezone.utc))
except Exception as e:
  print(f"ERROR: with timezone: {e}")
try:
  print(tstart.to_datetime())
except Exception as e:
  print(f"ERROR: without timezone: {e}")

ERROR: with timezone: name 'datetime' is not defined
ERROR: without timezone: ERFA function "d2dtf" yielded 1 of "unacceptable date (Note 6)"

Julian dates are equally applicable to the current time (and future). For example, this page was last edited:

import datetime
jd = astropy.time.Time(datetime.datetime.now(datetime.timezone.utc)).jd
print(f"It is now: {jd}")

It is now: 2459160.2741358937

Julian dates can be used to provide a consistent representation of geological time boundaries, albeit at a higher resolution than the actual boundaries can be determined. For example the Cenozoic 2 period ranges from 66 million years ago to modern time. This can be expressed in JD as:

cenozoic = (
  astropy.time.Time(-66E6, format='jyear').jd,
  astropy.time.Time(1950, format='jyear').jd,
)
print(f"The cenozoic period is JD {cenozoic[0]:.0f} to JD {cenozoic[1]:.0f}.")

The cenozoic period is JD -24104778955 to JD 2433282.

Positions in archeologic and ârecentâ geologic time are conventionally denoted in years, measured backwards from the present, which is fixed to 1950 when the precision requires it.

The start of the cenozoic is the Cretaceous-Paleogene Boundary (KBP, 3) determined to be 66.038 +- 0.025/0.049 Ma 4. That translates to JD:

c0 = astropy.time.Time(-66.038E6, format='jyear').jd
c1 = 365*1E6*0.025
c2 = 365*1E6*0.049
print(f"Cenozoic start = {c0:.0f} +- {c1:.0f}/{c2:.0f}")

Cenozoic start = -24118658455 +- 9125000/17885000
